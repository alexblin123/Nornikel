# НОРНИКЕЛЬ — Автоматизация анализа рудных шлифов

Система автоматической классификации медно-никелевых руд по микрофотографиям аншлифов. Определяет тип руды (рядовая / труднообогатимая / оталькованная), сегментирует минеральные фазы и поддерживает дообучение на экспертной разметке (Active Learning).

## Возможности

### 1. Интеллектуальный анализ шлифа
- Сегментация сульфидных фаз (OpenCV): разделение на крупные рядовые зёрна и тонкодисперсные труднообогатимые срастания
- Детекция талька нейросетью U-Net (бинарная сегментация)
- Тепловая карта вероятности присутствия талька
- Автоматический вердикт: Рядовая / Труднообогатимая / Оталькованная руда
- Экспорт результатов: CSV, PDF-паспорт образца, GeoJSON-контуры фаз

### 2. Пакетная обработка серий
- Через веб-интерфейс: загрузка ZIP-архива, сводная таблица по всем образцам, выбор любого образца для детального анализа
- Через CLI (без интерфейса, с логированием):

    python batch_process.py --input photos/ --output results/

  Результат: batch_results.csv, сводка и лог обработки

### 3. Active Learning — дообучение на экспертной разметке
- Геолог размечает минеральные фазы на интерактивном холсте прямо в UI (кисть/полигон, 3 фазы: обычные / тонкие / тальк)
- Разметка сохраняется как изображение, маски (png) и метаданные (json) в папку active_learning_dataset/
- retrain_talc.py дообучает U-Net на новых данных (transfer learning):
  - загружает текущие веса talc_unet.pth
  - дообучает на объединении исходного датасета и новой разметки
  - safe-save: модель обновляется только при росте Dice на валидации — защита от некорректной или случайной разметки пользователя

## Модель U-Net (детекция талька)

- Архитектура: U-Net для бинарной семантической сегментации талька
- Обучение: на GPU NVIDIA T4 (скрипт train_talc.py)
- Датасет: 42 размеченных микрофотографии шлифов (~220 МБ, хранится на GPU-сервере)
- Метрика: Dice-коэффициент на валидации ≈ 0.65–0.72 (зависит от train/val сплита)
- Файл модели: talc_unet.pth
- Тестирование: test_talc.py

Пайплайн ML:

    train_talc.py    — обучение U-Net с нуля (на GPU T4)
    test_talc.py     — оценка качества (Dice) на тестовой выборке
    retrain_talc.py  — дообучение на экспертной разметке (safe-save)

## Запуск

Требуется Python 3.11 (важно для совместимости streamlit-drawable-canvas).

    # создать виртуальное окружение
    python -m venv venv

    # активировать (Windows)
    venv\Scripts\activate

    # активировать (Linux/Mac)
    source venv/bin/activate

    # установить зависимости
    pip install -r requirements.txt

    # запустить веб-приложение
    streamlit run app.py

Дообучение модели (на GPU-сервере):

    python3 retrain_talc.py --base-data talc_dataset --new-data active_learning_dataset --epochs 30

## Структура проекта

    app.py                   — веб-интерфейс (Streamlit): анализ, разметка, пакетная обработка
    analyzer.py              — ядро анализа шлифа (OpenCV + U-Net)
    report.py                — экспорт отчётов (CSV / PDF / GeoJSON)
    batch_process.py         — пакетная обработка серии образцов (CLI)
    train_talc.py            — обучение U-Net с нуля (GPU)
    test_talc.py             — оценка качества модели (Dice)
    retrain_talc.py          — дообучение U-Net на экспертной разметке (Active Learning)
    talc_unet.pth            — обученная модель U-Net
    active_learning_dataset/ — экспертная разметка, собранная через UI
    requirements.txt         — зависимости проекта

Примечание: обучающий датасет talc_dataset/ (~220 МБ) не включён в репозиторий из-за размера и хранится на GPU-сервере.
